## CSS Units

- Lengths

  |:--------------------------|:----------------------------------------|
  | cm  (Centimeters)         |  em ( Font size of the parent )         |

  | pc  (Picas)               |  lh ( Line height of the element )      |
  | px  (Pixels)              |                                         |
  | pt  (Points)              |                                         |
